# Sentiq Clubes Fundadores Program

Version: 1.0

---

# Purpose

This document defines the Clubes Fundadores program.

The program is designed to create the first group of sports organizations that help validate, improve and establish Sentiq Org as a solution for athlete development.

---

# Program Vision

The first clubs that adopt Sentiq are not only customers.

They are strategic partners in building the future of athlete development.

The objective is to create a group of organizations that believe in:

- Athlete growth.
- Continuous improvement.
- Better coach-athlete relationships.
- Innovation in sports development.

---

# Strategic Objective

The Clubes Fundadores program has four main objectives:

- Validate Sentiq Org in real environments.
- Collect feedback from clubs, coaches and athletes.
- Create reference organizations.
- Build the foundation for future expansion.

---

# Why Clubes Fundadores

Early adoption is essential because Sentiq needs to understand:

- How clubs integrate the platform.
- How coaches use the information.
- How athletes respond.
- What processes create the most value.

---

# Clubes Fundadores Profile

The ideal founding clubs are organizations that:

- Value athlete development.
- Have structured youth categories.
- Are open to innovation.
- Have committed coaches.
- Want to improve their methodology.

---
