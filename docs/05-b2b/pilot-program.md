# Sentiq B2B Pilot Program

Version: 1.0

---

# Purpose

This document defines the Sentiq B2B pilot program.

The objective of the pilot is to validate Sentiq Org in real sports organizations, understand adoption challenges, collect feedback and create successful implementation cases.

---

# Pilot Philosophy

A pilot is not only a free trial.

It is a collaborative process between Sentiq and the organization.

The club helps validate the product while Sentiq helps the club improve its athlete development process.

---

# Pilot Objectives

The pilot has five main objectives:

- Validate product adoption.
- Understand coach experience.
- Measure athlete engagement.
- Identify product improvements.
- Create reference cases.

---

# Pilot Target

The ideal pilot organization:

- Has youth categories.
- Has committed coaches.
- Values athlete development.
- Is open to innovation.
- Can provide feedback.

---

# Initial Pilot Focus

Pilots can include organizations from any sport.

Priority outreach sports:

- Rugby.
- Hockey.
- Football.
- Basketball.

These markets are prioritized because they often offer:

- Strong development culture.
- Structured categories.
- Existing relationships.
- High alignment with Sentiq philosophy.

This prioritization is a go-to-market choice, not a product restriction.

---

# Pilot Scope

A pilot should have a clear and limited scope.

Recommended initial scope:

- One organization.
- One or more categories.
- Selected coaches.
- Selected athletes.

---

# Pilot Duration

Recommended duration:

8 to 12 weeks.

This allows enough time to:

- Complete onboarding.
- Create usage habits.
- Observe engagement.
- Collect feedback.

---

# Pilot Phases

---

# Phase 1

# Discovery And Setup

Objective:

Prepare the organization.

Activities:

- Understand club structure.
- Define categories.
- Identify coaches.
- Define objectives.

---

# Phase 2

# Onboarding

Objective:

Activate users.

Activities:

- Create organization.
- Configure users.
- Invite coaches.
- Invite athletes.

---

# Phase 3

# First Usage

Objective:

Create initial value.

Activities:

- Athletes complete evaluations.
- Athletes define objectives.
- Coaches review information.

---

# Phase 4

# Adoption

Objective:

Integrate Sentiq into development routines.

Activities:

- Encourage reflections.
- Support coach usage.
- Review feedback.

---

# Phase 5

# Evaluation

Objective:

Measure results.

Activities:

- Analyze metrics.
- Collect interviews.
- Identify improvements.

---

# Pilot Participants

---

# Club Leadership

Role:

- Support adoption.
- Communicate importance.
- Provide feedback.

---

# Coaches

Role:

- Use insights.
- Encourage athletes.
- Provide practical feedback.

---

# Athletes

Role:

- Complete evaluations.
- Define objectives.
- Participate consistently.

# Success Metrics

A successful pilot should evaluate:

---

# Adoption Metrics

Measure:

- Athletes activated.
- Coaches activated.
- Categories participating.

---

# Engagement Metrics

Measure:

- Evaluations completed.
- Objectives created.
- Returning users.

---

# Quality Metrics

Measure:

- Coach satisfaction.
- Athlete satisfaction.
- Perceived value.

---

# Product Feedback

Collect:

- Usability issues.
- Missing features.
- Improvement opportunities.

---

# Pilot Deliverables

At the end of the pilot:

Sentiq provides:

- Pilot analysis.
- Adoption report.
- Feedback summary.
- Recommendations.

---

The organization provides:

- Usage feedback.
- Testimonials if applicable.
- Improvement suggestions.

---

# Pilot Success Criteria

A pilot is successful when:

- Athletes understand the value.
- Coaches use Sentiq naturally.
- The club sees improvement opportunities.
- The organization wants to continue.

# Pilot Risks

---

## Low Internal Commitment

Without club support adoption decreases.

---

## Coach Resistance

The implementation must demonstrate value quickly.

---

## Low Athlete Participation

The experience must be simple and meaningful.

---

# Pilot Best Practices

---

## Start Small

A focused pilot creates better learning.

---

## Measure Everything

Usage data is essential for improvement.

---

## Listen To Users

Feedback from athletes and coaches drives product evolution.

---

# Long-Term Vision

The pilot program becomes the foundation for Sentiq's B2B expansion.

Each successful implementation creates:

- Knowledge.
- Improvements.
- References.
- New opportunities.

---

# Final Principle

The goal of a Sentiq pilot is not only to prove that the product works.

It is to prove that a better way of developing athletes is possible.
