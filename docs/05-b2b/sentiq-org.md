# Sentiq Org

Version: 1.0

---

# Purpose

This document defines Sentiq Org, the institutional version of Sentiq designed for sports organizations.

Sentiq Org extends the individual athlete experience into a structured environment where clubs, coaches and athletes can work together around athlete development.

---

# What Is Sentiq Org

Sentiq Org is the platform that enables sports organizations to implement a development methodology supported by technology.

It connects:

- Athletes.
- Coaches.
- Categories.
- Organizations.

The objective is to help clubs understand their athletes better and create a continuous development process.

---

# Vision

Sentiq Org transforms athlete development from an informal process into a structured methodology.

The platform helps organizations move from:
Intuition only

↓

Data + Reflection + Conversation

↓

Intentional Athlete Development

# Relationship Between Sentiq B2C And Sentiq Org

Sentiq B2C and Sentiq Org share the same athlete development philosophy.

The difference is the context.

---

# Sentiq B2C

Focused on:

- Individual athletes.
- Personal development.
- Self-reflection.
- Personal objectives.
- Individual evolution.

---

# Sentiq Org

Focused on:

- Organizations.
- Teams.
- Coaches.
- Development processes.
- Collective implementation.

# Product Principle

Sentiq Org does not replace the athlete experience.

It creates an ecosystem around it.

The athlete remains the owner of their development journey.

Sentiq Org has different user roles.

# Coach

- Using insights in conversations.
