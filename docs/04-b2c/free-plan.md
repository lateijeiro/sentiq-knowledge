# Sentiq Free Plan Strategy

Version: 1.1

Status: Current freemium split + conversion principles

---

# Purpose

This document defines the free experience in Sentiq's B2C model.

The free plan exists to:

- Introduce athletes to Sentiq
- Create development consistency
- Demonstrate product value
- Build trust
- Create natural Pro conversion opportunities

---

# Free Plan Philosophy

The free experience should be valuable by itself.

Sentiq should not create artificial limitations that prevent athletes from experiencing the core product.

The athlete should think:

"Sentiq helps me improve."

Before:

"I need to pay to use it."

---

# Strategic Principle

Free creates consistency.

Pro expands depth and personalization.

---

# Current Free Experience

A free athlete can:

- Create an account and complete onboarding
- Complete full and quick evaluations
- Write reflections (`hechoBien`, `mejorar`)
- Create and review objectives manually
- Create and manage Meta Foco
- Track progress, trends, streaks and IEA
- Use Sports Modes and notifications
- Use Red (follow, invite, feed, suggest objectives)
- Browse evaluation history from Home calendar

Free already delivers the Sentiq development loop.

---

# Current Pro Differentiation

Sentiq Pro unlocks deeper AI-assisted guidance:

| Capability | Free | Pro |
|------------|------|-----|
| Evaluations | Yes | Yes |
| Manual objectives | Yes | Yes |
| Meta Foco | Yes | Yes |
| Progress / IEA | Yes | Yes |
| Red | Yes | Yes |
| AI objective suggestions | No | Yes (`ia_objetivos`) |
| Weekly / pre-match Coach Sentiq | No | Yes (`coach_semanal`) |
| Monthly personal report | No | Yes (`informe_mensual`) |
| Learning Cards | No | Yes |

See [../03-product/premium.md](../03-product/premium.md).

Pro is not a conversational chatbot. It is a set of AI-generated outputs.

---

# How Athletes Get Pro (Current)

Effective Pro can come from:

- RevenueCat paid subscription
- Pilot / early adopter access
- Admin plan
- Organization grant
- Campaign benefit grant
- Unlock Pro benefit grant

Home prioritizes campaign card → Unlock Pro card → subscription upsell.

---

# Feature Evaluation Rule

A feature should be Pro only if:

1. It creates significant additional value
2. It does not prevent the athlete from experiencing the core development loop

---

# What Should Never Be Behind Pro

Do not block:

- Basic reflection / evaluations
- Ability to create and review objectives
- Core progress visibility (including IEA once eligible)
- Sports Modes and essential reminders
- The ability to improve through the product

---

# Conversion Psychology

Avoid fear-based conversion:

- "You lost access"
- "Your progress is locked"

Prefer growth-based conversion:

- "Get deeper personalized guidance"
- "Unlock monthly insights and Learning Cards"

Ideal conversion moment:

"I want to understand my development deeper."

Not:

"I cannot use the product anymore."

# Metrics

## Free Activation

% completing first evaluation.

## Free Retention

Return at Day 7 / 30 / 90.

## Pro Conversion

% of engaged free users becoming Pro (paid or grant).

## Revenue Quality

Long-term subscribers over short-term spikes.

---

# Free Plan Risks

## Too Much Free

If Pro has no meaningful additional value, conversion stays low.

## Too Little Free

If users cannot experience value, retention fails.

## Wrong Limits

Limits should protect Pro depth, not destroy the free habit.

# Final Principle

The free plan creates athletes who believe in Sentiq.

Pro helps those athletes go deeper.

The goal is not to restrict growth.

The goal is to create enough value that athletes choose to continue the journey.
