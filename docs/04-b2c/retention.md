# Sentiq B2C Retention Strategy

Version: 1.1

Status: Current retention drivers + planned improvements

---

# Purpose

This document defines how Sentiq creates long-term athlete engagement.

Retention is not about making athletes use the app more.

Retention is about making Sentiq valuable throughout the athlete development journey.

---

# Retention Philosophy

Most apps compete for attention.

Sentiq competes for commitment.

The athlete returns because:

"I am building myself."

Not because:

"I need to maintain a streak."

Streaks and reminders may support consistency. They are not the purpose of the product.

---

# Core Retention Principle

Value must increase over time.

The longer an athlete uses Sentiq:

- The more history they create
- The better Coach Sentiq context becomes
- The clearer IEA and trends become

---

# The Sentiq Retention Loop (Current)

```
Experience (training / match)
   ↓
Evaluation / Reflection
   ↓
Objectives
   ↓
Practice
   ↓
Progress (IEA, trends, reports, Learning Cards)
   ↓
New Evaluation
```

# Retention Drivers

## 1. Personal Progress (Current)

Product elements:

- Evaluation history
- Objective completion
- IEA, confidence, tier, trend
- Monthly report (Pro)
- Learning Cards (Pro)

## 2. Consistency Through Routine (Current)

Desired behaviors:

- Post-training evaluation
- Post-match evaluation
- Objective review in the next evaluation
- Weekly / monthly insight review (Pro)

Supported by Sports Modes and notifications — not by a standalone habits module.

## 3. Personalization (Current)

Sources:

- Previous evaluations
- Objectives and Meta Foco
- Sport / profile
- Mode and schedule

## 4. Meaningful Moments (Current)

Examples:

- After training: "I learned something about myself"
- Before competition: PrepBanner + Coach Sentiq preparation
- After a month: monthly report / Learning Card

## 5. Community (Current, limited)

Current:

- Red follower network
- Shared evaluations (respecting privacy)
- Follower-suggested objectives
- Home invitation CTA and pending invite banner

Planned:

- Broader athlete stories
- Challenges
- Stronger recognition

## 6. Pro Paths That Reward Engagement (Current, flag-gated)

- Unlock Pro program (activity → Pro grant)
- Campaign Pro access for partner cohorts
- Paid Pro after value is established

# User Lifecycle

## Stage 1 — New User (first days)

Goal: experience value.

Key actions:

- Complete onboarding
- First evaluation
- First objective

Risk: purpose not understood.

## Stage 2 — Developing User (first weeks)

Goal: create routine.

Key actions:

- Regular evaluations
- Objective follow-up
- Mode/schedule notifications working

Risk: experience feels repetitive or noisy.

## Stage 3 — Engaged Athlete (months)

Goal: deepen relationship.

Key actions:

- Review IEA / trends
- Maintain Meta Foco alignment
- Use Pro insights if available

Risk: progress feels invisible.

## Stage 4 — Advocate

Goal: organic growth.

Behavior:

- Recommends Sentiq
- Invites teammates (Red / word of mouth)
- Shares experience

# Notifications Strategy

Notifications should create value, not interruption.

Good direction:

- Post-event reflection reminders tied to schedule
- Pre-match preparation windows
- Off-mode gentle daily check-in

Avoid:

- "You haven't opened Sentiq"
- Shame for broken streaks
- Fake urgency

Every notification should answer: Why now? Why does it matter? What action?

# Retention Metrics

## Activation

First evaluation completed.

## Engagement

WAU / MAU, evaluation frequency, objective activity.

## Retention

Day 7 / 30 / 90.

## Consistency

Weekly evaluations, returning athletes, objective reviews.

## Product Health

Multiple evaluations, maintained objectives, recommendations, Pro conversion after value.

---

# Churn Signals

- First evaluation only
- No objective created
- No return after onboarding
- Notifications disabled and no return
- No perceived progress

## Prevention (Current-aligned)

- Mode-aware reminders
- Preparation moments before matches
- Progress visibility (IEA)
- Pro depth after trust (reports / Learning Cards)
- Unlock Pro for engaged free users (when enabled)

Avoid solving churn with:

- More notifications alone
- Gamification without purpose
- Artificial urgency

---

# Retention And Pro

Pro conversion should happen after value is established.

Ideal:

"I want more depth."

Not:

"I need access to continue."
