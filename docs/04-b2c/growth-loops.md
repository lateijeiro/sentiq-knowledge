# Sentiq B2C Growth Loops

Version: 1.1

Status: Current loops + Planned loops

---

# Purpose

This document defines the growth mechanisms that allow Sentiq to acquire new athletes through product value, community and relationships.

Growth loops are continuous systems where one successful user action creates opportunities for future growth.

---

# Growth Philosophy

Sentiq should not depend only on paid acquisition.

The strongest long-term engine should be:

Athletes who believe in Sentiq inviting other athletes.

Until that referral program ships, growth still compounds through product value, clubs, campaigns and community.

---

# Core Growth Loop (Strategy)

```
Athlete joins Sentiq
   ↓
Experiences personal improvement
   ↓
Shares experience / invites
   ↓
Another athlete discovers Sentiq
   ↓
New athlete starts journey
   ↓
Community grows
```

---

# Growth Principles

1. Product value creates growth
2. Trust drives adoption
3. Community compounds
4. Do not force virality before retention

---

# Loop 1 — Product Value Loop (Current)

## Description

An athlete gets value from evaluations, objectives and progress, then returns.

## Flow

```
Evaluation
   ↓
Objective
   ↓
Progress / IEA
   ↓
Return after next event
```

## Why It Matters

Without this loop, every other growth loop fails.

Supported by Sports Modes, notifications, Home calendar and Pro insights.

---

# Loop 2 — Club / Coach Loop (Current)

## Description

A club or coach introduces Sentiq; athletes use it; the organization sees value; more categories join.

## Flow

```
Club / coach adopts Sentiq Org or recommends the app
   ↓
Athletes join (org invite or organic)
   ↓
Athletes evaluate and set objectives
   ↓
Coach sees Pulse / player insights
   ↓
More rosters / teams adopt
```

## Why It Matters

One relationship can generate many athletes.

May include org Pro grants for members.

---

# Loop 3 — Campaign / Partner Loop (Current)

## Description

A partner, event or pilot shares a campaign code; athletes join and receive a defined Pro benefit; some stay as retained users.

## Flow

```
Partner / event launches campaign
   ↓
Athletes redeem code
   ↓
Pro grant (capacity-limited)
   ↓
Athletes experience Pro depth
   ↓
Retention and word of mouth
```

See [campaigns.md](./campaigns.md).

This is partner distribution, not personal referral attribution.

---

# Loop 4 — Red Community Loop (Current, weak acquisition)

## Description

Athletes invite teammates to follow them, share evaluations and suggest objectives.

## Flow

```
Athlete uses Red
   ↓
Invites teammate
   ↓
Follow relationship forms
   ↓
Shared development support
```

## Limitation

Red currently strengthens engagement more than measured acquisition.

It does not attribute registrations or grant Pro for invites.

See [referrals.md](./referrals.md).

---

# Loop 5 — Unlock Pro Engagement Loop (Current, flag-gated)

## Description

Free athletes complete meaningful activity and earn temporary Pro access, which deepens value and retention.

## Flow

```
Evaluations + completed objectives (+ later stage requirements)
   ↓
Unlock Pro progress
   ↓
Pro benefit grant
   ↓
Deeper AI guidance
   ↓
Higher retention / possible paid conversion later
```

See [../03-product/premium.md](../03-product/premium.md).

---

# Loop 6 — Content Education Loop (Current strategy)

## Description

Content helps athletes; some discover Sentiq; their journeys create more stories and content.

Surfaces: website, Instagram, LinkedIn, blog.

---

# Loop 7 — Athlete Referral Loop (Planned)

## Description

An athlete invites another athlete with attribution because Sentiq helped them.

## Trigger examples

- Several evaluations completed
- Objective achieved
- Evolution insight / report

## Incentive Philosophy

Primary motivation: "I want my teammates to improve too."

Possible later recognition or limited Pro time for quality activations.

Avoid forced unlock walls.

---

# Viral Coefficient

The objective is not immediate virality.

The objective is sustainable organic growth.

Important metric:

How many new active athletes come from existing athletes, clubs and partners?

---

# Growth Metrics

## Acquisition

New users, source, conversion.

## Invite Systems (split)

- Red invites sent / accepted
- Campaign joins / capacity usage
- Org invitations accepted
- Planned: personal referral conversions

## Organic Growth

Organic registrations, direct traffic, brand searches.

## Community

Active athletes, active clubs, ambassador participation.

---

# Product Requirements For Growth

## Shareable Moments (partially current)

Learning Cards can be shared; evaluations can be public to followers.

Continue creating moments athletes want to share without ranking pressure.

## Social Proof

Athlete stories, club adoption, testimonials.

## Easy Invitation

Current: Red invites, campaign links, org invites.

Planned: one-tap personal referral link.

---

# What Sentiq Should Avoid

- Forced virality ("Invite 5 friends to unlock core features")
- Empty sharing without emotional value
- Growth before retention
- Mixing campaign codes with personal referrals in analytics

---

# Growth Priorities (Current Stage)

1. Create product love
2. Strengthen evaluation consistency
3. Use clubs + campaigns thoughtfully
4. Build first community (early adopters + Red)
5. Design and ship quality referral attribution
6. Scale paid acquisition only after retention proof

---

# Final Principle

The best growth strategy for Sentiq is creating athletes who genuinely believe:

"My teammates should experience this too."
