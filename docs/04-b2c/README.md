# Sentiq B2C Product Strategy

Version: 1.1

Status legend used across this folder:

- **Current**: implemented in the product (may be feature-flagged in production)
- **Planned**: intentional next work
- **Strategy**: durable principles that guide decisions

---

# Purpose

This document defines the direct-to-athlete strategy for Sentiq.

B2C is the relationship between Sentiq and individual athletes through sentiq-app.

The objective is not only acquiring users.

The objective is creating athletes who believe Sentiq helps them improve and continue using it over time.

---

# B2C Vision

Sentiq starts with one athlete.

One reflection.

One improvement.

One consistent development routine.

The B2C journey is about proving that athletes independently value personal development and want a tool to support their growth.

---

# Primary Objective

Create a community of athletes who use Sentiq as part of their development routine.

Success is measured by:

- Active athletes
- Consistent evaluations
- Objective creation and review
- Long-term retention
- Willingness to recommend and pay

---

# B2C Target User

Primary user:

- Young athlete
- Approximately 14–23 years old
- Any sport
- Priority early outreach: rugby, hockey, football and basketball

Secondary influence channels:

- Coaches and clubs (often through Sentiq Org or partnerships)
- Teammates through Red

---

# Athlete Motivation

Athletes join Sentiq because they want to:

- Improve
- Understand themselves better
- Build confidence
- Become more consistent
- Achieve personal objectives

They do not join because they want:

- Another tracker
- A comparison network
- Another obligation

---

# B2C Value Proposition

Sentiq helps athletes improve through a simple loop:

Reflect → Understand → Set objectives → Practice → See progress → Reflect again

---

# Business Model

The B2C model is freemium.

## Free (Current)

Enough value to create a real development habit:

- Evaluations (full and quick)
- Objectives (manual)
- Meta Foco
- Progress and IEA
- Sports Modes and notifications
- Red (follower network)

## Sentiq Pro (Current)

Deeper personalized guidance:

- AI objective suggestions
- Weekly / pre-match Coach Sentiq preparation
- Monthly personal report
- Learning Cards

See [../03-product/premium.md](../03-product/premium.md) and [free-plan.md](./free-plan.md).

## Pricing reference (Current)

Store prices come from RevenueCat offerings. Reference targets used in planning:

- Monthly: $6.99 USD
- Annual: $59.99 USD

Treat store-configured offerings as the source of truth for live prices.

# Current B2C Growth Mechanisms

| Mechanism | Status | Role |
|-----------|--------|------|
| Clubs / coaches / partnerships | Current strategy + Org product | Concentrated acquisition |
| Early Adopters (first 100) | Current (flag-gated auto-assign) | Validation community + lifetime Pro |
| Campaigns (partner codes) | Current (flag-gated grants) | Partner / event Pro distribution |
| Unlock Pro program | Current (flag-gated) | Activation → Pro conversion path |
| Red invites | Current | Social follow network (not referral rewards) |
| Content (site, Instagram, LinkedIn) | Current | Awareness |
| Athlete-to-athlete referral rewards | Planned | Organic invite attribution + incentives |
| Paid ads at scale | Planned (after retention proof) | Scale |

# Acquisition Philosophy

Grow through trust first:

- Coaches
- Teammates
- Clubs
- Sports communities

Paid acquisition should follow product love, not precede it.

See [user-acquisition.md](./user-acquisition.md).

---

# Activation Goal (Current)

Ideal activation after account creation:

1. Complete onboarding (profile, mode, schedule)
2. Complete first evaluation
3. Create first objective
4. Return for a second evaluation

Onboarding itself prepares context. First value usually happens in the first evaluation, not inside onboarding screens.

See [onboarding.md](./onboarding.md).

---

# Retention Philosophy

Retention is based on value, not addiction.

Users return because:

- They learn something
- They see progress (IEA, trends, reports)
- They have objectives
- Notifications remind them at the right moment

See [retention.md](./retention.md).

---

# B2C Growth Stages

## Stage 1 — Validation (Current focus)

Prove athletes love the product.

Metrics: first users, activation, first evaluations, feedback.

## Stage 2 — Habit Formation (Current)

Create regular usage.

Metrics: WAU, evaluation frequency, objective usage.

## Stage 3 — Community Growth (Partially current)

Expand through athletes, clubs, campaigns and Red.

Metrics: organic acquisition, campaign joins, org invites, future referral conversions.

## Stage 4 — Monetization (Current infrastructure)

Convert engaged athletes into paying users.

Metrics: Pro conversion, RevenueCat subscriptions, grant-assisted retention, LTV.

---

# What B2C Must Prove

Before scaling aggressively:

- Athletes return without external pressure
- Athletes create meaningful evaluations
- Athletes value progress visibility
- Athletes recommend Sentiq
- Athletes are willing to pay (or accept Pro through meaningful grants)
